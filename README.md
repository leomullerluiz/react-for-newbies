# React for newbies
Based on:
- [Install Tailwind CSS with Vite](https://tailwindcss.com/docs/guides/vite)

## Recommended IDE Setup
> git clone https://github.com/leonan-luiz/react-for-newbies.git

> cd react-for-newbies

> npm install

> npm run dev

To run in local network:
> npm run dev -- --host
